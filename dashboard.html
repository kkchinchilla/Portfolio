Launch Project

Freelance writers need to manage multiple aspects of their professional lives, from time management, project management, to financial management. The purpose of this project is to create a dashboard that allows writers to monitor all aspects of their work.

Target Audience

Freelance novelists aged 30-45 who have been in the industry for a while and who often have to deal with multiple projects at the same time.

- Number of completed projects

• Weekly Activity
